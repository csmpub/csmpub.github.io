페이지 이동, 경로 등 설정

페이지 이동 설정

힌트Seperators
  • sidebar:하부 섹션(section) 또는 항목(item) 사이에 text: "---"가 있는 경우 보이지 않음
website:
  sidebar:
    contents:
      - section: "section one"
        contents:
          - href: documents.qmd
          - ...
      - text: "---"
      - section: "section two"
        contents:
          - href: ...
          ...
프로젝트 설정(_quarto.yml)
website:
  page-navigation: ture   # or false
폴더 설정(_metadata.yml)
page-navigation: true   # or false
문서 설정(yml 헤더)
---
page-navigation: false  # or true
---

페이지 경로 설정

프로젝트 설정(_quarto.yml)
website:
  bread-crumbs: false
문서 설정(yml 헤더)
---
title: Page without breadcrumbs
bread-crumbs: false
---

페이지 꼬리말 설정

프로젝트 설정(_quarto.yml)
website:
  page-footer:
    left: "단순 텍스트 문자열"
    center: ...
    right:
      - icon: github
        href: https://github.com
page-footer 옵션

background foreground border

website:
  page-footer:
    border: flase   # default - true
    background: light   # default - no background color
힌트foreground color

지정하지 않으면 background와 대비되는 색으로 자동 지정됨

처음으로 설정

프로젝트 설정(_quarto.yml)
website:
  back-to-top-navigation: ture   # or false
문서 설정(yml 헤더)
---
page-navigation: false  # or true
---
